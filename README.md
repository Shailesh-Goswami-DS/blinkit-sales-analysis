# 📊 Blinkit Sales Analysis

## 📌 Problem Statement
Blinkit is a quick-commerce grocery delivery platform. The objective of this project is to analyze sales performance, customer purchasing patterns, and outlet efficiency to uncover actionable business insights that can help improve revenue and operational efficiency.

---

## 🎯 Objectives
- Identify top-performing product categories based on total sales.
- Analyze sales trends across different outlet types and locations.
- Evaluate outlet performance based on size and establishment year.
- Detect high-revenue regions and product segments.
- Provide data-driven recommendations for business growth.

---

## 📊 Dataset Overview
- Dataset contains sales records including item details, outlet information, item visibility, and sales revenue.
- Key Features:
  - Item Type
  - Item Fat Content
  - Outlet Type
  - Outlet Location Type
  - Outlet Size
  - Item MRP
  - Sales

---

## 🛠 Tools & Technologies Used
- **Python** (Pandas, NumPy) – Data cleaning & preprocessing
- **Matplotlib & Seaborn** – Data visualization
- **Power BI** – Interactive dashboard creation
- **SQL** – Data querying (if applicable)

---

## 🔎 Exploratory Data Analysis (EDA)
- Handled missing values and inconsistent data.
- Performed categorical and numerical feature analysis.
- Visualized sales distribution by outlet type and product category.
- Compared revenue across different outlet sizes and locations.

---

## 📈 Key Insights
- 🏆 Supermarket Type1 generated the highest overall sales.
- 📍 Tier 3 cities showed strong revenue contribution.
- 🛒 Regular fat items contributed more revenue compared to low-fat items.
- 📅 Older established outlets showed consistent sales performance.
- 💰 Higher MRP items contributed significantly to total revenue.

---

## 🚀 Business Impact
- Identified high-performing product segments to optimize inventory planning.
- Provided insights to improve marketing strategies in high-revenue regions.
- Suggested focusing on Tier 3 locations for expansion opportunities.
- Enabled data-driven decision-making through interactive dashboards.
